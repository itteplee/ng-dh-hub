# ng-dh-hub

本仓库用于归档和发布多份独立的 HTML 页面。这些页面以静态文件的形式存放，方便直接通过 GitHub Pages 或其他静态托管服务访问。

## 项目简介

ng-dh-hub 是一个轻量级的页面集合仓库。仓库内不涉及特定域名、网站或商业用途，仅作为 HTML 文件的整理与存储空间。每个页面均为独立、自包含的 HTML 文件，适合用作演示、存档或快速分享。

## 目录说明

仓库根目录下包含以下结构：

- `index.html`：仓库首页，列出所有已归档页面的索引。
- `pages/`：存放所有独立 HTML 页面的子目录。每个页面通常以单独的文件名存在，如 `page-example.html`。
- `assets/`：可选目录，用于存放页面所需的共用资源（如 CSS、JS、图片等），具体是否使用视页面需要而定。

## 页面归档说明

- 所有页面均以 `.html` 后缀保存，编码为 UTF-8。
- 每个页面应保持独立，不依赖外部非仓库内的资源。
- 页面内容不涉及敏感、非法或侵权信息。
- 归档的页面可能不定期更新或移除，建议使用者自行备份。
